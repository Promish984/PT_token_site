# Fungible Token Project (React + Rust/Axum)

## Run Backend
```bash
cd backend
cargo run
# listens on http://127.0.0.1:3001
```

## Run Frontend
```bash
cd frontend
npm install
npm run dev
# opens http://localhost:3000 (proxied to backend on 3001)
```
